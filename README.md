# Pyloa - (Py)thon (l)ocation (o)ptimization (a)lgorithms

Pyloa is a collection of methods for solving, (i) location problems in the Euclidian plane, (ii), location problems on a network (graph) and, (iii) discrete facility location problem. The package is primarily intended for teaching purposes. See the package's [documentation](https://andreasklose.github.io/pyloa/) for further information.

Note that this is a preliminary release that still misses a couple of modules, which will be added at a later time.
